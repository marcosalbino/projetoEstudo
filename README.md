## Aplicação de Busca de Atletas

**Descrição:**

Este aplicativo web simples permite pesquisar por atletas ou esportes em uma base de dados pré-definida. Ao digitar um termo na caixa de pesquisa, os resultados relevantes são exibidos em tempo real, incluindo o nome do atleta, uma breve descrição e um link para mais informações.

**Funcionalidades:**

* **Pesquisa:** Permite pesquisar por palavras-chave em títulos, descrições e tags de cada atleta.
* **Resultados dinâmicos:** Os resultados da pesquisa são atualizados em tempo real à medida que o usuário digita.
* **Links para mais informações:** Cada resultado possui um link que direciona o usuário para uma página externa com mais detalhes sobre o atleta.

**Tecnologias Utilizadas:**

* **HTML:** Estrutura básica da página, definindo os elementos da interface do usuário.
* **CSS:** Estilização da página, definindo a aparência visual dos elementos.
* **JavaScript:** Lógica da aplicação, responsável por:
    * Capturar o termo de pesquisa.
    * Filtrar os dados da base de dados.
    * Renderizar os resultados na página.
* **JSON:** Formato utilizado para armazenar os dados dos atletas (no arquivo `dados.js`).

**Como usar:**

1. **Clonar o repositório:** Clone este repositório para sua máquina local usando o Git.
2. **Abrir o arquivo HTML:** Abra o arquivo `index.html` em um navegador web.
3. **Realizar uma pesquisa:** Digite o nome de um atleta ou esporte na caixa de pesquisa e pressione Enter.

**Estrutura de arquivos:**

* `index.html`: Arquivo principal da aplicação.
* `style.css`: Arquivo de estilos CSS.
* `dados.js`: Arquivo JSON contendo os dados dos atletas.
* `app.js`: Arquivo JavaScript com a lógica da aplicação.
